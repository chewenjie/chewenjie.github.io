二、运行过程

编写代码的文件是以.java为后缀的源文件，通过编译生成以.class为后缀的字节码文件

  • Hello.java(源文件) => javac Hello.java 编译程序

  • Hello.class(字节码文件) => java Hello 运行程序

什么是编译：
  1. 有了可编译的java源程序(例如Hello.java源文件)
  2. 在该源文件目录下，通过javac.exe编译工具对java源文件进行编译
  3. 如果程序编译没有错误，将没有任何提示，并在当前目录下会生成一个字节码文件
什么是运行：
  1. 有了可执行的java程序(例如Hello.class字节码文件)
  2. 通过java.exe运行工具对字节码文件进行执行

三、注意事项

  1. java源文件以.java为扩展名
  2. 源文件的基本组成部分是类(class)
  3. java应用程序的执行入口是main()方法：public static void main(String[] args) { … }
  4. java语言严格区分大小写，以’;’表示每条语句的结束符号
  5. 大括号都是成对出现的，缺一不可，养成先写{ }后再在里面写代码的习惯
  6. 一个源文件中只能有一个public类，其类名与文件名必须相同，其他非public类的个数不限
  7. main()方法也可以写在非public类中，然后指定运行的非public类，这样入口方法就是非public类的main()方法
  8. 对每一次修改的源文件都需要保存后，重新进行编译，然后再运行

五、注释

用于注释说明和解释程序代码的文字就是注释

  1. 注释提高了代码的可读性
  2. 注释是一个程序员必须要具有的良好的编程习惯

java程序的注释一共分为三种类型：

符号 注释类型 使用方法
// 单行注释 // 这是一个单行注释
/* */ 多行注释 /* 这是一个多行注释 */
/** */ 文档注释 /** 这是一个文档注释 */

注意：注释的文字，不会被JVM解释执行，且多行注释里面不允许有多行注释的嵌套

六、转义字符

由于计算机的某些特殊字符无法直接用字符表示

就可以通过转义字符”\“的方式表示，也就是将原字符的含义转为其他含义

转义字符 代表意思
\‘ 单引号（’）
\“ 双引号（”）
\\ 反斜杠（\）
\r 回车
\n 换行
\t 横向制表符
