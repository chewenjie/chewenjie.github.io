Java学习笔记第二章-变量

一、变量

变量就是申请内存来存储值，也就是说，当创建变量的时候，需要在内存中申请空间

内存管理系统根据变量的类型为变量分配存储空间，分配的空间只能用来存储该类型的数据

变量的注意事项：
  1. 变量必须先声明后使用
  2. 变量在同一个作用域内不能重名
  3. 变量 = 数据类型 + 变量名 + 值（三要素）

二、数据类型

java的数据类型分为基本数据类型和引用数据类型

基本数据类型：

Java语言提供了八种基本数据类型，六种数字类型（四个整数型，两个浮点型），一种字符类型，还有一种布尔型

关键字 数据类型 占用内存 取值范围
byte 字节型 1个字节 -128 ~ 127
short 短整型 2个字节 -32768 ~ 32767
int 整型 4个字节 -2147483648 ~ 2147483647
long 长整型 8个字节 -2^63 ~ 2^63-1
float 浮点型 4个字节 -3.403E38 ~ 3.403E38
double 双精度浮点型 8个字节 -1.798E308 ~ 1.798E308
char 字符型 2个字节 0 ~ 655535
boolean 布尔型 1个字节 true、false
引用数据类型（先了解即可，后面会详解）：

在Java语言中引用数据类型指向一个对象，指向对象的变量是引用变量

这些变量在声明时被指定为一个特定的类型，变量一旦声明后，其数据类型就不能被改变了

关键字 数据类型
class 类
interface 接口
[] 数组

三、注意事项

整数类型：
  1. 各整数类型有固定的范围和字段长度，不受具体OS（操作系统）的影响，以保证java程序的可移植性
  2. 整数常量默认为int型，声明long型常量必须在数值后面加上’l’或’L’
  3. bit（位）：计算机中的最小存储单位；byte（字节）：计算机中的基本存储单元；1byte = 8bit
浮点数类型：

  1. 与整数类型类似，浮点数类型也有固定的范围和字段长度，不受具体OS的影响，以保证java程序的可移植性

  2. 浮点数类型默认为double型，声明float型常量必须在数值后面加上’f’或’F’

  3. 浮点数常量有两种表现形式：

    • 十进制形式：如5.12、5.12.0f、.512（当整数部分为0时可以忽略，但小数点必须要）
    • 科学计数法形式：如5.12e2、5.12E-2（’e’或’E’都可以表示幂）

  4. 通常情况下，应该使用double型的浮点数，因为它比float型的浮点数精确度更高

  5. 浮点数使用陷阱：（2.7）和（8.1 / 3）的比较

    • 因为（8.1 / 3）得到的是一个接近2.7的浮点数，而不是2.7，二者并不相等
    • 于是在做判断条件if(2.7 == (8.1 / 3))时，无法得到想要的结果
    • 正确的写法：if(Math.abs(2.7 - (8.1 / 3)) < 0.000001(自定精度))
    • Math.abs()方法是取两个数的差值，这样当差值非常小，小到自定的精度时，就认为两者相等
字符类型：
  1. 字符常量是用单引号括起来的单个字符
  2. 允许使用转义字符来将其后的字符转变为特殊字符型常量
  3. 在java中，char类型的本质是一个整数，在输出时，输出的是Unicode码对应的字符
  4. char类型是可以进行运算的，相当于一个整数，因为它有对应的Unicode码值
  5. 字符类型本质的探讨：
    • 存储：’a’ => Unicode码值：97 => 二进制：01100001 => 进行存储
    • 读取：二进制：01100001 => Unicode码值：97 => ‘a’ => 进行读取
布尔类型：
  1. 只允许取值为true或false
  2. 适用于逻辑运算，一般用于程序流程的控制
  3. 不可以用0或非0的整数替代false或true，这点和C语言不同

四、类型转换

自动类型转换：
  1. 在进行赋值或运算时，精度小的类型会自动转换为精度大的数据类型
  2. 当byte、short或char三者进行计算时首先会将其数据类型转换为int类型
  3. 自动提升原则：
    • 表达式结果的类型自动提升为操作数中最大的类型
  4. 数据类型按精度大小排序为：
    • char > int > long > float > double
    • byte > short > int > long > float > double
    • char和byte及short 之间不可转换，boolean不参与类型的自动转换
强制转换类型：
  1. 是自动转换类型的逆过程，将容量大的数据类型转换为容量小的数据类型
  2. 使用时要加上强制转换符’()’，但可能会造成精度损失或数据溢出，格外需要注意
  3. 强制转换符只针对最近的操作数有效，往往会使用小括号提升优先级
  4. char类型可以保存int类型的常量值，但不能保存int类型的变量值，此时需要强制转换
  5. byte和short类型在进行运算时，当作int类型处理

五、标识符

在java语言中，对各种变量、方法和类等命名时使用的字符序列称为标识符

凡是可以自己起名字的地方都叫标识符，例如：int num = 100；其中的num就叫标识符

标识符的命名规则：

  1. 由26个英文字母的大小写、0到9数字、_下划线或$符号组成

  2. 不能以数字开头，且标识符不能是关键字或保留字，但是可以包括关键字和保留字

    • 关键字: 被java语言赋予了特殊含义，有专门的用途，用户不可将其作为标识符
    • 保留字: 现java语言尚未使用，但以后的版本可能会作为关键字使用，用户不可将其作为标识符

  3. java语言严格区分大小写，长度无限制，标识符不能包含空格

标识符的命名规范：
  1. 包名：所有字母都小写。如：aaa.bbb.ccc
  2. 类名、接口名：所有单词的首字母大写。如：AaaBbbCcc（大驼峰）
  3. 方法名、变量名：第一个单词的首字母小写，从第二个单词开始每个单词的首字母大写。如：aaaBbbCcc（小驼峰或驼峰）
  4. 常量名：所有字母都大写，每个单词之间用下划线连接。如：AAA_BBB_CCC
  5. 更多的命名规范，推荐看《阿里巴巴Java开发手册》
