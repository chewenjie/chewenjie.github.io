一、运算符

运算符是一种特殊的符号，用以表示数据的运算、赋值和比较等

二、算术运算符

算术运算符是对数值类型的变量进行运算的，在java程序中使用的非常多

运算符 运算 范例 结果
+ 正号 a = +7; a = 7
- 负号 a = -11; a = -11
+ 加 a = 9 + 9; a = 18
- 减 a = 8 - 6; a = 2
* 乘 a = 4 * 6; a = 24
/ 除 a = 9 / 3; a = 3
% 取模（取余） a = 5 % 2; a = 1
++ 自增（前：先自增后运算） a = 2; b = ++a; b = 3
++ 自增（后：先运算后自增） a = 2; b = a++; b = 2
– 自减（前：先自减后运算） a = 2; b = –a; b = 1
– 自减（后：先运算后自减） a = 2; b = a–; b = 2
+ 字符串相加 s = “hello” + “world” s = “helloworld”
注意事项：

  1. 在进行运算时，精度小的数据类型会自动转换为精度大的数据类型

  2. %取模运算符的运算本质：

    • 当a为整数时：a % b = a - a / b * b
    • 当a为浮点数时：a % b = a - (int)a / b * b

  3. ++自增运算符或–自减运算符在前: 先运算后取值；在后: 先取值后运算

三、关系运算符

关系运算符的结果都是boolean型，也就是要么是true，要么是false

运算符 运算 范例 结果
== 相等 6 == 9 false
!= 不等 6 != 9 true
> 大于 6 > 9 false
< 小于 6 < 9 true
>= 大于等于 6 >= 9 false
<= 小于等于 6 <= 9 true
instanceof 是否是类的对象 6 instanceof int true

四、逻辑运算符

逻辑运算符用于连接多个条件（多个关系表达式），最终的结果也是一个boolean值

运算符 运算 范例 规则
& 逻辑与 a & b 当a和b同时为true，则结果为true，否则为false
&& 短路与 a && b 当a和b同时为true，则结果为true，否则为false
| 逻辑或 a | b 当a和b有一个为true，则结果为true，否则为false
|| 短路或 a || b 当a和b有一个为true，则结果为true，否则为false
^ 逻辑异或 a ^ b 当a和b不同时，则结果为true，否则为false
! 取反（非运算） !a 当a为true，结果为false；当a为false，结果为true
注意事项：

  1. &和&&的使用区别（一般使用短路与&&，效率高）

    • &&短路与: 如果第一个条件为false，则第二个条件不会判断，最终结果为false

    • &逻辑与: 不管第一个条件是否为false，第二个条件都要做判断，&逻辑与比&&短路与效率低

  2. |和||的使用区别（一般使用短路或||，效率高）

    • ||短路或: 如果第一个条件为true，则第二个条件不会判断，最终结果为true
    • |逻辑或: 不管第一个条件是否为true，第二个条件都要做判断，|逻辑或比||短路或效率低

五、赋值运算符

赋值运算符就是将某个运算后的值，赋给指定的变量

  • 基本赋值运算符: =
  • 复合赋值运算符: +=、-=、*=、/=、%= 等

对+=分析，其他复合赋值运算符类似

复合赋值运算符会进行类型转换: int a += 5; 等价a = a + 5; 底层 a = (int)(a + 5);

六、三元运算符

条件表达式 ? 表达式1 : 表达式2

  • 如果条件表达式为true，运算后的结果是表达式1
  • 如果条件表达式为false，运算后的结果是表达式2
  • 表达式1和表达式2都要为可以赋值给接收变量的类型(或可以自动转换，或强制转换)

七、运算符的优先级

java语言中运算符的优先级共分为14级，其中1级最高，14级最低

在同一个表达式中运算符优先级高的先执行，运算符优先级及结合性表如下：

优先级 运算符 结合性
1 ()、[]、{} 从左向右
2 !、+、-、~、++、– 从右向左
3 *、/、% 从左向右
4 +、- 从左向右
5 <<、>>、>>> 从左向右
6 <、<=、>、>=、instanceof 从左向右
7 ==、!= 从左向右
8 & 从左向右
9 ^ 从左向右
10 | 从左向右
11 && 从左向右
12 || 从左向右
13 ? : 从右向左
14 =、+=、-=、*=、/=、&=、=、^=、~=、<<=、>>=、>>>= 从右向左

记忆口诀: 单目乘除位关系，逻辑三目后赋值

  • 单目：单目运算符+ –(负数) ++ – 等
  • 乘除：算数运算符* / % + -
  • 位：位移运算符<< >> >>>
  • 关系：关系运算符> < >= <= == != 等
  • 逻辑：逻辑运算符&& || & | ^
  • 三目：三目运算符 ? :
  • 后：无意义，仅仅为了凑字数
  • 赋值：赋值= += -= *= /= 等

其实在实际的开发中，不需要去记忆运算符的优先级别，也不要刻意的使用运算符的优先级别

对于不清楚优先级的地方使用小括号辅助进行优先级管理，更方便写代码，也便于代码的阅读和维护
